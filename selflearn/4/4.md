# 分割コンパイルの方法
1. ヘッダファイル(`hpp`)ファイルに関数宣言する。
2. ソースファイル(`cpp`)でインクルードする。

```cpp:module.hpp
void show_value(int value);
```

```cpp:module.cpp
#include "module.hpp"
#include <iostream>

void show_value(int value)
{
    std::cout << "value = " << value << std::endl;
}
```

```cpp:main.cpp
#include "module.hpp"

int main()
{
    show_value(42);
}
```

```
$ g++ -c -o module.o module.cpp
$ g++ -o a.out main.cpp module.o
$ ./a.out
value = 42
```
